# Gym Class Booker

A Selenium automation project that:

- Logs into the gym website
- Books available classes
- Joins waitlists when classes are full
- Verifies bookings
- Handles stale elements and timeouts using retry logic

## Features

- Explicit waits
- Retry mechanism
- Booking verification
- Waitlist support
- Exception handling

## Tech Stack

- Python
- Selenium
- ChromeDriver

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
